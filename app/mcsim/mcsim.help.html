MC Sim

This program was created in 2016 on the basis and with the command set of a Windows 8.1 Desktop micro controller simulator which was used at the BBZW Sursee, Switzerland. As this simulator had issues running on some windows 10 systems, this program was created for the test preparation.

Many things are configurable, but there is only one finished command set until now because the class moved on to the much more complicated x86 assembler after the above mentioned test. For more command sets, open an issue on GitHub and I'll help where I can.

Keyboard Shortcuts

A small set of keyboard shortcuts are supported for the program editor. All these shortcuts only work while the editor text field is focused.

Key Combo Action
Ctrl+Enter Compile program
Ctrl+S Save program
Ctrl+R Save program and replace it if it already exists
